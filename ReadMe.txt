A Platformer Shooter Prototype using Unity with C#

Inputs:
A: Left
D: Right
F: Shoot
Space: Jump

Score:
Skeleton give 1 point
Ghost give 3 points
Spectral Ghost give 5 points

Variant Mode:
Greater rate of fire
Less HP
More enemies
Faster enemies

To complete:
Multiplayer
Add music and SFX
Update score on UI (only on debug)
On timer end display score and give option to return to menu
